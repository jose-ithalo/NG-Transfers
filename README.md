# NG-Transfers
> Este projeto se trata de uma API que possibilita aos usuários cadastrados da NG realizarem <br>transferências internas entre si e visualizarem seu saldo.

### Campos de preechimento:
#### Users
* username
* password

### Login
* username
* password

### account
* value
* recipient

### transactions
* date
* cashOut
* cashIn
